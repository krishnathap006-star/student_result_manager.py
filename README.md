# Student Manager App

A simple Python command-line application to manage student records.

## Features
- Add Student
- View Students
- Check Pass/Fail Result
- Menu-driven interface

## Technologies Used
- Python 3
- Dictionary
- Loops
- Conditional Statements

## How to Run

```bash
python student_manager.py
```

## Sample Menu

1. Add Student
2. View Students
3. Check Result
4. Exit
